## 📝 專案概述與研究動機
本專案探討如何透過程式化生成（Programmatic Generation）技術，解決傳統數學教育中「抽象幾何空間感難以建立」的問題。
專案採用 Python 與 3blue1brown 開發的數學圖形引擎 **Manim**，將複雜的靜態幾何定理與解題邏輯，轉化為具備嚴謹時序控制的動態視覺化動畫。此系統不僅展示了圖學渲染的應用，更體現了如何以演算法思維拆解教學步驟，降低學習者的認知負荷（Cognitive Load）。

## 🛠️ 技術特點與工程實作
* **座標幾何與圖學控制**：精準定義二維空間座標點，透過 OOP（物件導向）設計將點、線、多邊形物件化（`VGroup`, `Polygon`），便於進行矩陣變換與動畫插值。
* **動態時序與併發處理**：大量運用 `AnimationGroup`、`lag_time` 與 `run_time` 控制多個幾何物件的渲染併發與非同步動畫，確保視覺引導符合人類的認知邏輯。
* **數學邏輯程式化**：將「證明過程」轉化為程式邏輯，例如動態計算並繪製角平分線、定義精確的弧度（`Angle.from_three_points`）以驗證平行與同位角性質。

## 📐 核心展示模組 (Core Modules)
本專案包含四個循序漸進的幾何動態證明腳本，涵蓋了從基礎屬性到複雜定理的視覺化推導：

### 1. 平行線與同位角/內錯角動態推導 (`PointsAndLinesScene - 模組一`)
* **實作內容**：透過定義特定的視角與截線，動態展示平行線間的角度關係。
* **視覺化亮點**：使用顏色漸變（Color Transformation）與相機聚焦（`FocusOn`），引導視線比對 `FAE` 與 `ECD` 等角度，直觀驗證 `BC // EF` 等平行性質。

### 2. 動態尋找與證明等腰三角形 (`PointsAndLinesScene - 模組二`)
* **實作內容**：演示在一個平行四邊形（ABCD）中，利用角平分線（AF）構造出多個等腰三角形的過程。
* **視覺化亮點**：結合波浪特效（`ApplyWave`）強調截線，並透過閃爍（`Flash`）提示內錯角與對頂角的相等關係，最終將找出的三個等腰三角形以不同的透明度色塊（`fill_opacity`）高亮顯示。

### 3. 平行四邊形內角平分線交會定理 (`PointsScene - 模組三`)
* **實作內容**：程式化證明「平行四邊形四個內角的平分線，其交點會形成一個矩形」。
* **視覺化亮點**：將代數運算視覺化。在畫面上動態呈現 `2綠 + 2黃 = 180度` 推導至 `1綠 + 1黃 = 90度` 的過程。透過遮罩與幾何切割，證明內側交出的多邊形四個角皆為直角，最終渲染出中心的藍色矩形。

### 4. 同底同高與多邊形面積變換 (`PointAndLineScene - 模組四`)
* **實作內容**：解決幾何題型中最抽象的「重疊面積比較」問題，比較平行四邊形內兩個跨度不同的三角形面積。
* **視覺化亮點**：不依賴死記硬背，而是透過程式動態繪製「同底」與「同高」的輔助線與直角標記（`elbow=True`）。利用白色區塊作為「減法遮罩」，動態演繹「等面積扣除不同大小重疊區域」的邏輯。

**前置需求：**
* Python 3.8+
* Manim Community Edition (`manim`)
* FFmpeg (用於影片編碼與輸出)
* LaTeX 分發版 (用於數學公式與文字渲染)
