# UPAD 自動化出題腳本 (Auto-Question-Generator)

## 📝 專案簡介
本專案為一款基於 Python (Selenium) 開發的自動化腳本，旨在取代繁瑣且重複的網頁出題步驟。透過自動化操作實現批量出題，大幅節省人工操作時間並提升整體工作效率。

## 🚀 操作流程

請依照以下步驟執行自動化出題：

1. **設定出題排程**
   將準備要出題的 UPAD 項目寫入 `schedule` (Excel 檔) 中。
   > ⚠️ **注意**：請仔細核對輸入內容，避免錯字；「學年」欄位請務必正確選擇（例如：`112上` 或 `112下`）。

2. **確認排程編號**
   記下您在 `schedule` 表格中所對應的「**數字編號**」。

3. **執行主程式**
   打開 `execute` 執行檔（或腳本），將程式碼中的 `schedule_id` 參數修改為剛剛記下的數字編號，確認後執行檔案。

4. **完成與清理進度**
   當系統跳出「**完成了**」的提示時，代表出題排程已全數結束。此時，請手動刪除目錄下的 `current_mission` 檔案。

---

## 💡 進階功能：中斷與接續出題

專案具備「斷點續傳」的機制：
`current_mission` 檔案的作用是**記錄系統上一次出題的進度**。如果您在出題過程中需要暫停腳本以執行其他任務，只要**保留**該檔案不刪除，下次重新執行程式時，系統便會自動從上次中斷的地方接續出題。

---

## ⚠️ 注意事項與常見問題

* **單次出題數量上限**：當一次排程的出題數 **大於 200 題** 時，系統可能會跳出紅色錯誤訊息中斷。此為預期現象，您只需**再次執行 `execute`**，程式即會讀取 `current_mission` 並接續往後出題，無需擔心重複執行。
* **檔案完整性**：為確保程式能正常運作，本資料夾中**除了 `current_mission` 以外，請勿刪除任何其他檔案或資料夾**。
