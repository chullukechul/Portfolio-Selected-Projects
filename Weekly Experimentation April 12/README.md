實作一-二元分類應用:癌症判別
實作目標
本週進度著重於將 Logistic Regression、Backpropagation。藉由威斯康辛乳癌資料集建立二元分類器，中途遇到不少問題，透過 AI 輔助工具逐一解惑來釐清並鞏固所學新知。

所學新知與克服的疑問
資料狀態保存與實驗控制： 理解了 StandardScaler 必須實例化為物件，是為了記憶訓練集的均值與標準差，確保測試集套用相同的特徵縮放基準。同時，明確了設定 random_state 對於控制實驗變因、確保模型表現具備可重現性的重要性。

維度對齊： 釐清了 Numpy 1D Array (N,) 與 PyTorch 2D 行向量 (N, 1) 在底層運算上的嚴格差異。若未主動使用 .view(-1, 1) 強制升維，PyTorch 的廣播機制將導致 Loss 函數進行錯誤的矩陣擴展，引發難以察覺的錯誤。

PyTorch 物件導向架構 (OOP) 哲學： 掌握了繼承 nn.Module 時，super().__init__() 的必要性在於初始化底層的梯度追蹤與計算圖字典。理解了 __init__ 專注於註冊具備權重的網路層，而 forward 則專注於建構無狀態的資料流向與啟動函數（如 Sigmoid）。

記憶體管理與梯度機制： 發現在 PyTorch 中必須手動執行 optimizer.zero_grad()，因為其預設為梯度累加機制（此特性可用於小顯卡模擬大 Batch Size）。而在推論評估階段，則需透過 with torch.no_grad(): 結界關閉自動微分引擎，避免浪費記憶體。

試錯實驗
為了驗證理論，我主動對超參數進行了破壞性實驗：

發現一：Sigmoid 的「梯度安全網」特性
我將 Learning Rate 暴力調大至 50.0，原預期模型訓練失敗。然而實驗結果顯示，準確率依然高達 99.12%（僅 Loss 數值異常停滯）。經數學推導發現，在單層 Logistic Regression 中，Sigmoid 的飽和特性將預測誤差鎖定在 (0, 1) 之間，加上特徵已標準化，使得梯度存在上限，模型反而因極度武斷而切分出正確的決策邊界，觀察到老師課堂上沒有提及的特性。

發現二：MSE 在分類問題的結構性崩潰
李老師有分享 MSE 在分類問題會因極端值而導致訓練困難的問題，我拔了 Sigmoid，將損失函數硬改為線性迴歸的 MSE。實驗結果顯示，在正常的 Learning Rate (0.05) 下，由於資料集本身具備高度的線性可分性，模型準確率仍有 96%，相較於使用 Cross-Entropy 的 97.37% 只出現輕微效能退步。印證 MSE 會為了迎合極端值而過度拉扯決策邊界，部分犧牲了邊界模糊地帶的準確度但「資料集本身的線性可分性極高」(此處採用 AI 的見解) 可能導致準確率依舊維持良好水平。

實作練習二-質數判別
受李宏毅老師課堂提及 Joel Grus 以機器學習判別 FizzBuzz（3 與 5 的倍數）實驗之啟發，設計了一個注定失敗的對照實驗：訓練 DNN 來判別質數與合數。此實驗來探尋神經網路在盲區。

AI 建議優化的部分： 將 2 到 1023 的十進位整數轉換為 10-bit 的二進位陣列作為模型輸入。

考量： 若直接輸入連續實數，質數的高頻震盪與極度非平滑特性會迫使神經網路扭曲其決策邊界，導致碎裂。

優化預期： 透過二進位編碼，試圖讓模型轉為學習位元間的布林邏輯閘關係。

實驗過程與發現
發現一：Overfitting
實驗 A（高容量）： 建立包含兩個隱藏層（各 64 個神經元，約 4800 個參數）的 DNN。

結果： 訓練集準確率 100%，測試集準確率僅約 74.63%。

見解： 在此數值區間內，合數佔比約 83.3%（盲猜的正確比例）。模型在測試集表現遠低於盲猜，證明其龐大的參數量使其淪為死記硬背，無泛化能力。

實驗 B（低容量）： 試圖抑制死背，將隱藏層神經元數量大幅縮減至 4 個（僅剩 69 個參數）。

結果： 訓練集準確率降為 88.62%，測試集準確率反而微幅回升至 79.02%。

見解： 當模型容量被嚴格限制，無法背下所有答案時，發生某部分特徵學習。模型僅學會了最表層的膚淺特徵（可能學到：二進位最後一碼是 0，即合數），利用此單一條件篩掉多數錯誤以提升準確率，但本質上仍未學會質數的規律，總體還是優於 A 實驗。

發現二：比較 Sigmoid 與 ReLU
實驗設定： 依據裡李老師課堂理論，多層網路使用 Sigmoid 易引發梯度消失。因此我將參數/隱藏層神經元與上述 A 實驗相同並把隱藏層的 ReLU 全數替換為 Sigmoid，原預期模型將因無法更新權重而停滯。

結果： 訓練並未卡死，Training Loss 依然順利收斂至 0.0003，且達成 100% 訓練準確率。

見解： 網路深度太淺： 模型僅有 2 層隱藏層，連鎖律造成的梯度衰減程度上可接受。
